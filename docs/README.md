# 文档目录

本目录用于存放 GeWeHub 的中文优先文档，包括:

- 产品设计
- 架构设计
- API 契约
- 标准消息结构
- 消息 Markdown 渲染标准
- 部署说明
- 调研记录

后续文档应优先保持可执行、可验证，避免只写概念。

## 当前核心文档

- `标准消息结构-v1.md`：标准消息 JSON 结构、类型枚举、GeWe 映射和投递模型。
- `消息Markdown渲染标准-v1.md`：`renderedMd` 后续提案，当前契约仍以结构化 JSON 和 `renderedText` 为准。
- `HTML发送功能开发方案-v1.md`：新增 `type=html` 发送、HTML 托管、管理端和 Hermes 插件/CLI 接入的后续开发目标。
